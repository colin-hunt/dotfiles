# WezTerm Keybindings

## Panes

| Action | Keys |
|:-------|:-----|
| Split right | `Cmd` `Ctrl` `→` |
| Split down | `Cmd` `Ctrl` `↓` |
| Navigate | `Cmd` `Option` `Arrow` |
| Rotate clockwise | `Cmd` `Option` `r` |
| Rotate counter-clockwise | `Cmd` `Option` `Shift` `r` |
| Close pane | `Cmd` `w` |

## Tabs

| Action | Keys |
|:-------|:-----|
| New tab | `Cmd` `t` |
| Move left | `Cmd` `Shift` `←` |
| Move right | `Cmd` `Shift` `→` |
| Navigate prev | `Cmd` `Shift` `[` |
| Navigate next | `Cmd` `Shift` `]` |
| Go to tab 1-9 | `Cmd` `1-9` |
| Close tab | `Cmd` `Shift` `w` |

## Modifier Pattern

| Modifier | Context |
|:---------|:--------|
| `Cmd` | Basic actions (close pane, new tab) |
| `Cmd` `Ctrl` | Pane splitting |
| `Cmd` `Option` | Pane navigation & rotation |
| `Cmd` `Shift` | Tab management |
